Twitter scraping tool!


In this website you're able to enter a certain #hashtag, and look some interesting data about the latest 200 tweets.

After entering a hashtag, the website will show you the following analyses in a visual way:

  • Sentiment analysis – How the positivity, negativity and neutrality has distributes
  • Subjectivity analysis – How the objectivity and subjectivity has distributed
  • Location analysis – Where the tweets have been tweeted
